HTML Table Cheat Sheet

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML elements.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The data in the table.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Tag Name Description
colspan Column Span Defines how many columns a <td> should span.
rowspan Row Span Defines how many rows a <td> should span.
